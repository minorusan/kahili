# Investigation: ScoreMaster A2B end target null
## TLDR
This is a high-frequency error log rather than a crash, but it indicates the ScoreMaster fly-up animation is attempting to play when the lobby icon target no longer exists. The A2B controller logs an error when the end transform for `ScoreMasterIcon` is null, which likely happens when the lobby icon view is destroyed during operation teardown.
## Risk Assessment
Low crash risk (logged error only), but it signals a lifecycle bug that can suppress the fly-up animation and generate noisy Sentry alerts in production.
## Release Branch
v3.34.6
## Release Version
3.34.6
## Error Summary
`[A2BController] -> Can not play preset A2B Preset Score Master: end transform ScoreMasterIcon is null - skip effect...` emitted from `ninja.Components.A2b.A2BController.PlayPreset` during `ScoreMasterRewardsGiveawaySequenceOperation.PlayFlyUpAnimationAsync`.
## Root Cause
`ScoreMasterRewardsGiveawaySequenceOperation` registers the `ScoreMasterIcon` target once in its constructor, but the fly-up animation can run after the lobby icon view has been destroyed (stack traces show teardown via `Zenject.MonoKernel:OnDestroy` and operation discard). When the Unity `Transform` is destroyed it evaluates to null, so `_locator.GetTargetTransform(A2BTarget.ScoreMasterIcon)` returns null and `A2BController.PlayPreset` logs an error.
## Affected Code
- `Assets/Scripts/A2B/A2BController.cs` (null end transform error log)
- `Assets/Scripts/LiveOps/ScoreMaster/Lobby/ScoreMasterRewardsGiveawaySequenceOperation.cs` (PlayFlyUpAnimationAsync uses `A2BTarget.ScoreMasterIcon`)
## Suggested Fix
In `ScoreMasterRewardsGiveawaySequenceOperation.PlayFlyUpAnimationAsync`, guard against a missing/invalid lobby icon target before calling `PlayPreset`:
- Re-register the target right before playing and skip the effect if `_lobbyIconView.Transform` is null or not active.
- Optionally tie the operation’s cancellation to the lobby icon’s destroy token to stop the sequence when the view is disposed.
Concrete example:
- If `_lobbyIconView.Transform == null` then `return` (or log a warning), otherwise `_a2BController.RegisterTarget(A2BTarget.ScoreMasterIcon, _lobbyIconView.Transform)` and proceed.
